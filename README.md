# DCA Management using Digital & AI Solutions

## Problem
FedEx manages overdue customer accounts using external DCAs through manual tools like Excel and emails, causing delays and lack of transparency.

## Solution
A centralized digital platform that automates case allocation, prioritization using AI logic, and provides dashboards for tracking DCA performance.

## Features
- Automated case prioritization
- Basic AI recovery prediction
- Centralized dashboard
- Role-based DCA interaction

## Tech Stack
- Python
- HTML
- Pandas

## How to Run
1. Install dependencies
2. Run app.py
3. Open index.html
